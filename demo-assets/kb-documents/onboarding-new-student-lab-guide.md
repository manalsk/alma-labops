# Onboarding Guide: New Student Lab User

## Purpose
This document helps new students understand how to use the lab safely and how to interact with ALMA LabOps.

## First Week Checklist
1. Complete safety briefing.
2. Review biohazard waste SOP.
3. Review package intake SOP.
4. Learn storage locations: room temperature shelves, chemical cabinet, refrigerator, -20°C freezer, and -80°C freezer if authorized.
5. Learn how to search inventory in ALMA LabOps.
6. Learn how to update assigned tasks.
7. Ask a researcher before moving or consuming unfamiliar materials.

## Student Permissions in ALMA LabOps
Students can:
- View and search inventory
- View onboarding documents and SOPs
- Upload package photos
- Run extraction if permitted
- Complete assigned tasks
- Submit purchase suggestions

Students cannot:
- Approve purchases
- Finalize inventory from package intake
- Manage users
- Manage vendors
- View financial summaries
- Delete inventory items

## Common Questions

### What should I do if I use the last item?
Create a task request or notify a researcher so inventory can be updated.

### What should I do if I receive a package?
Upload a clear photo in Incoming Packages and notify the assigned researcher.

### What should I do if I am unsure where something belongs?
Do not guess. Ask a researcher or check ALMA LabOps inventory location records.
