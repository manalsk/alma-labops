# SOP: Cold Storage and Freezer Inventory

## Purpose
This SOP explains how cold-chain materials should be stored, labeled, tracked, and monitored in ALMA LabOps.

## Storage Categories
- Room temperature
- Refrigerated: 2–8°C
- Frozen: -20°C
- Ultra-low freezer: -80°C
- Liquid nitrogen storage, if applicable

## Procedure
1. Every cold-chain item must have a storage condition recorded in inventory.
2. New cold-chain packages must be unpacked immediately after arrival.
3. Items marked “Keep Frozen” must not remain on the bench longer than necessary.
4. Each item must be labeled with item name, catalog number, lot number if available, date received, and owner/project.
5. Update ALMA LabOps inventory after placing the item in storage.
6. Freezer locations should follow the format: Freezer name / Rack / Box / Position.

## Example
Anti-GFP Antibody, catalog A-11122:
- Storage condition: Keep Frozen
- Suggested location: Freezer A / Rack 2 / Box 4
- Handling note: Minimize freeze-thaw cycles

## Alerts
If a freezer temperature alarm occurs, notify the PI and senior researcher immediately.

## Responsible Role
Researcher or operations researcher.
